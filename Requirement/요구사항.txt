1. 2인용 
-> 각각의 우주선은 다른색
-> 각각의 점수 기록
2. 게임 난이도 설정 가능
-> 쉬움, 중간 어려움
-> 각각의 레벨은 다른 적, 총알, 내 우주선의 속도가 달라야함
-> 새로 만들 게임 난이도는 기존의 게임과는 다른 난이도를 가져야함
-> 다른 난이도 게임을 통해 얻은 점수는 개별적 저장되어야함
3. 더 좋은 인터페이스
-> esc키로 게임 멈추기
-> 점수 리셋하는 방법 만들기
4.추가요구사항
-> 최고 기록 깨지면 점수 빨갛게 표시하기
-> 상단에 현재 레벨과 난이도를 표시하기
-> 최고기록 등수 10등까지 난이도도 표시하기 
-> 배경 음악을 삽입하기
-> 난이도 선택 화면에서 뒤로 가기 기능을 추가하기
-> 1인용과 2인용 점수 분리하기
